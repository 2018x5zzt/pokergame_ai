# AI 斗地主直播项目

## 项目愿景
打造一个 AI 斗地主直播娱乐平台：AI 玩家通过 CCCC 框架对战斗地主，前端可视化展示，通过 OBS 推流到抖音直播，观众可通过刷礼物影响比赛，形成互动娱乐直播新模式。

## 核心目标
1. 开发完整的斗地主游戏引擎（规则、牌型、AI决策）
2. 设计精美的前端可视化界面（适配直播画面比例）
3. 实现 OBS 推流到抖音直播的完整技术链路
4. 建立观众礼物互动影响比赛的机制
5. 策划赛事体系提升直播热度
6. 确保合规运营，规避封号风险

## 交付物（DoD）
- 10 份专业级策划文档，覆盖规则、设计、技术、运营全维度
- 所有文档存放在 `docs/` 目录下
- 文档之间逻辑自洽、可直接指导开发实施

## 技术栈（建议）
- 游戏引擎：TypeScript/Python
- 前端展示：React + Canvas/WebGL
- AI框架：CCCC 多智能体协作
- 推流：OBS Studio + RTMP
- 弹幕/礼物：抖音开放平台 API / 第三方弹幕姬
- 部署：本地运行 + OBS 窗口捕获

## 合规要点
- 2026年2月1日起《直播电商监督管理办法》要求 AI 内容必须标识
- 抖音虚拟主播需备案
- 棋牌类直播需避免赌博嫌疑
- 主播健康分机制需关注
