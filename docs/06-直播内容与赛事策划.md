## 一、直播内容定位

### 1.1 频道定位
**"AI大乱斗"** —— 全网首个 AI 斗地主对战直播频道

| 维度 | 定位 |
|------|------|
| 内容类型 | AI 娱乐 + 棋牌竞技 + 观众互动 |
| 核心标签 | #AI斗地主 #人工智能 #棋牌直播 #互动直播 |
| 差异化 | 不是人打牌，是 AI 打牌；观众不是看客，是"上帝" |
| 调性 | 轻松搞笑、科技感、有梗、有悬念 |

### 1.2 内容价值主张
1. **娱乐价值**：AI 的"骚操作"和"神仙打架"天然有趣
2. **互动价值**：刷礼物影响比赛，观众有参与感和掌控感
3. **社交价值**：支持不同 AI 角色，形成粉丝阵营对立
4. **知识价值**：AI 解说分析策略，观众能学到牌技

## 三、AI 解说系统

### 3.1 解说架构

采用"主解说 + 角色自评"双层解说模式：

| 层级 | 角色 | 触发时机 | 内容风格 |
|------|------|----------|----------|
| 主解说 | 独立解说AI | 每次出牌、关键节点 | 客观分析、策略点评、悬念制造 |
| 角色自评 | 对战AI自身 | 出牌后、被压制时 | 人格化吐槽、情绪表达、挑衅互动 |

### 3.2 解说触发规则

| 触发事件 | 解说内容 | 优先级 | TTS时长上限 |
|----------|----------|--------|-------------|
| 叫地主 | 分析手牌强度、叫分合理性 | 中 | 5秒 |
| 普通出牌 | 简短点评出牌意图 | 低 | 3秒 |
| 关键出牌（大牌/拆牌） | 策略深度分析 | 高 | 8秒 |
| 炸弹/火箭 | 激动解说 + 倍数提醒 | 最高 | 6秒 |
| 连续PASS | 分析局势僵持原因 | 中 | 5秒 |
| 剩余≤3张牌 | 紧张倒计时解说 | 高 | 4秒 |
| 胜负揭晓 | 总结本局亮点、MVP评选 | 最高 | 10秒 |
| 春天/反春天 | 特殊激动解说 | 最高 | 8秒 |

### 3.3 解说文案模板

#### 3.3.1 开局阶段
```
模板1: "好的，新的一局开始了！{玩家}叫了{分数}分拿下地主，底牌是{底牌}，这手牌{评价}！"
模板2: "三位AI选手蓄势待发，让我们看看这局谁能笑到最后！"
模板3: "{玩家}直接叫3分！看来这把牌相当有信心啊！"
```

#### 3.3.2 对局阶段
```
模板1: "{玩家}打出{牌型}！{评价}，{对手}还剩{数量}张牌，压力来了！"
模板2: "炸弹！！！{玩家}直接炸弹伺候！倍数翻倍变成{倍数}倍了！"
模板3: "连续两人不出，{玩家}拿到自由出牌权，这波节奏很关键！"
模板4: "{玩家}只剩{数量}张牌了！农民们要注意了！"
```

#### 3.3.3 结算阶段
```
模板1: "{胜方}获胜！本局最终倍数{倍数}倍，{地主}积分{变化}！"
模板2: "春天！{玩家}一张牌都没出就结束了，太惨了！"
模板3: "精彩的一局！{MVP}本局表现最佳，打出了{亮点}！"
```

### 3.4 TTS 技术要求

| 参数 | 要求 |
|------|------|
| 语音引擎 | 支持多音色（男声解说 + 3个AI角色各自音色） |
| 延迟要求 | 从触发到播放 ≤ 1.5秒 |
| 语速 | 正常语速1.0x，紧张时刻1.2x |
| 情绪表达 | 支持平静、兴奋、紧张、搞笑等语调变化 |
| 并发控制 | 同一时刻只播放一条解说，新解说可打断低优先级解说 |
| 音量层级 | 解说音量 > BGM音量，出牌音效与解说不冲突 |

### 3.5 TTS 内容审核要求

TTS 解说文案虽以模板为主，但包含动态变量填充，需纳入内容安全审核体系（详见 `07-抖音合规与风控 §六 内容安全审核机制`）。

| 审核阶段 | 审核对象 | 审核方式 | 说明 |
|----------|----------|----------|------|
| 预审（开播前） | 解说模板文案库 | 人工审核全量模板 | 确保模板本身不含敏感词、赌博暗示 |
| 实时审核（直播中） | 模板变量填充值 | 敏感词过滤引擎自动校验 | 变量值（玩家名、牌型描述等）需通过白名单校验 |
| 复审（直播后） | TTS 输出日志 | 抽检回听 + 日志分析 | 每周抽检 ≥ 2 场直播的 TTS 输出记录 |

> **合规要点**：所有 TTS 播报内容中，涉及倍数的表述统一使用「能量加成」而非「翻倍」；涉及积分的表述统一使用「虚拟战绩分」而非「筹码」。具体敏感词替换规则见 `07-抖音合规与风控 §4.3`。


## 五、特殊玩法模式

### 5.1 玩法模式一览

| 模式名称 | 触发方式 | 规则变化 | 适用场景 |
|----------|----------|----------|----------|
| 限时出牌 | 挑战赛环节 | 每手牌必须在5秒内出完 | 增加紧张感 |
| 礼物加倍局 | 观众累计礼物达标 | 本局所有倍数×2 | 刺激礼物消费 |
| 明牌局 | 特殊赛事 | 所有手牌公开可见 | 纯策略对抗 |
| 观众选地主 | 中场互动投票 | 观众投票决定谁当地主 | 增强参与感 |
| 炸弹狂欢局 | 周末特别场 | 每人额外获得1个炸弹 | 高倍数刺激 |
| 反转局 | 不定期 | 牌面大小反转（3最大，大王最小） | 新鲜感 |

### 5.2 限时出牌模式详细规则

```
基础规则:
- 每手牌出牌时限: 5秒
- 倒计时从轮到该玩家时开始
- 超时处理: 自动选择"不出"(跟牌时) 或 自动出最小单牌(自由出牌时)
- 画面展示: 倒计时进度条 + 最后2秒闪烁警告

AI适配:
- AI决策时间限制为3秒（留2秒给动画和网络延迟）
- 超时兜底: 随机选择一个合法出牌
```

### 5.3 观众选地主模式详细规则

```
流程:
1. 发牌完成后，暂停叫地主流程
2. 画面展示三个AI角色 + 投票按钮
3. 观众通过发送弹幕 "1"/"2"/"3" 投票
4. 投票时间: 30秒
5. 得票最多的AI直接成为地主（叫分固定为2分）
6. 平票时随机从平票AI中选一个

画面设计:
- 三个AI头像并排展示
- 实时票数柱状图动态增长
- 倒计时 + 投票音效
- 结果揭晓时聚光灯效果
```

### 5.4 明牌局模式详细规则

```
规则变化:
- 所有玩家手牌对观众完全可见
- AI之间仍然互相不可见（保持博弈性）
- 解说可以深度分析每步决策的优劣
- 适合教学向内容，帮助观众理解AI策略

画面调整:
- 三方手牌同时展示在画面上
- 当前出牌方手牌高亮放大
- 解说面板增加"最优出牌建议"对比
```

## 七、观众运营与互动设计

### 7.1 观众分层运营

| 观众层级 | 定义 | 运营策略 |
|----------|------|----------|
| 路人观众 | 首次进入直播间 | 新手引导弹窗、AI角色介绍、免费互动体验 |
| 活跃观众 | 连续观看≥3天 | 专属弹幕颜色、参与投票权、粉丝团入口引导 |
| 核心粉丝 | 加入粉丝团 | 粉丝专属互动、优先投票权、粉丝排行榜 |
| 大R用户 | 累计礼物≥500元 | VIP弹幕置顶、专属感谢动画、影响力加权 |

### 7.2 弹幕互动玩法

| 弹幕指令 | 功能 | 触发条件 |
|----------|------|----------|
| "加油+角色名" | 给指定AI加油，触发加油特效 | 任意时刻 |
| "预测+角色名" | 预测本局赢家，猜对有积分 | 每局开始前 |
| "1"/"2"/"3" | 投票选地主（观众选地主模式） | 投票环节 |
| "炸弹" | 触发弹幕炸弹特效（纯视觉） | 出现炸弹时 |
| "666" | 触发666弹幕飘屏 | 精彩操作时 |

### 7.3 粉丝阵营系统

每个观众可选择支持一个 AI 角色，形成三大阵营：

| 阵营 | 代表色 | 阵营口号 | 专属福利 |
|------|--------|----------|----------|
| 🔥 烈焰军团 | 红色 | "全场我最强！" | 红色弹幕、火焰特效 |
| 🧊 冰山联盟 | 蓝色 | "冷静制胜。" | 蓝色弹幕、冰晶特效 |
| 🎭 戏精帮 | 紫色 | "惊不惊喜？" | 紫色弹幕、彩带特效 |

阵营机制：
- 每周统计各阵营总礼物值，阵营排名第一的获得"本周最强应援团"称号
- 阵营成员在对应AI获胜时获得双倍观众积分
- 月度阵营总决赛：三大阵营PK，输的阵营AI下月开场要"道歉"

### 7.4 观众预测系统

```
流程:
1. 每局开始前，弹幕发送"预测+角色名"参与预测
2. 预测窗口: 叫地主阶段结束后10秒内
3. 预测正确: +10观众积分
4. 连续预测正确: 额外奖励（3连+20, 5连+50, 10连+200）
5. 观众积分用途: 兑换专属弹幕样式、参与抽奖

展示:
- 预测比例实时显示（如: 烈焰哥45% | 冰山姐35% | 戏精弟20%）
- 预测正确时全屏撒花 + 恭喜弹幕
```

### 8.4 BGM 与音效规划

| 场景 | BGM风格 | 音量 | 说明 |
|------|---------|------|------|
| 开场/收场 | 电子/科技感 | 70% | 营造氛围 |
| 热身赛 | 轻松爵士 | 40% | 不抢解说 |
| 正式赛 | 紧张节奏 | 50% | 配合对局节奏 |
| 决胜局 | 史诗/燃向 | 60% | 烘托高潮 |
| 中场休息 | 轻音乐 | 50% | 放松过渡 |
| 炸弹时刻 | 短促重低音 | 80% | 冲击力 |
| 春天时刻 | 胜利号角 | 80% | 仪式感 |

> **交叉引用说明**：上表为**节目级 BGM 规划**，定义每个直播环节（开场、热身赛、正式赛等）使用的背景音乐风格。在单局对局内部，BGM 会根据游戏阶段自动切换，具体的**对局级 BGM 映射**（等待/发牌 → 轻松中国风、正常出牌 → 节奏轻快、紧张时刻 → 紧张悬疑、结算 → 胜/败音乐）详见 `03-特效与动画设计 §13.1`。两个层级的关系为：节目级 BGM 决定当前环节的音乐基调和音量，对局级 BGM 在此基调下进行游戏阶段内的动态切换。
